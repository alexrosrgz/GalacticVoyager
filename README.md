# GalacticVoyager

A 3D space combat and exploration game built with Three.js. Pilot your spacecraft through the solar system, fly between planets, and fight off waves of enemy ships.

## How to Play

- **WASD** - Thrust / strafe
- **Mouse** - Aim (yaw & pitch)
- **A / D** - Roll
- **Shift** - Boost
- **Left Click** - Fire lasers

## Getting Started

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually `http://localhost:5173`).

## Tech Stack

- [Three.js](https://threejs.org/) - 3D rendering
- [Vite](https://vitejs.dev/) - Build tool & dev server
- Vanilla JavaScript (ES modules)
